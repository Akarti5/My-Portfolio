My experiences include

FINEX

I developed FINEX, an online banking management system, as a personal project. Built with PHP and MySQL, it features client management, transaction tracking, loan handling, and secure account operations. This project allowed me to deepen my understanding of backend logic, data security, and web app architecture.

Product Stock Management

I created a Product Stock Management System using Vue.js for the frontend and PHP with PostgreSQL for the backend. The app includes real-time inventory tracking, product CRUD operations, and category management. This project strengthened my skills in building responsive UIs with Vue and working with relational databases and RESTful APIs.

School Management

I developed a School Management System using PHP and MySQL, designed to handle student enrollment, class schedules, grading, and teacher assignments. This project helped me apply database-driven design, user role management, and backend functionality to solve real-world administrative tasks in education.
